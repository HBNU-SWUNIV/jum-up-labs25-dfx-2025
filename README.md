## Project Background
깻잎은 연중 수확이 가능한 주요 시설 엽채류이며, 현재까지도 대부분의 수확 과정이 숙련 인력의 수작업에 의존하고 있다. 장시간 반복으로 인한 신체적 피로와 인력 부족문제가 있으며, 효율적이고 정밀한 자동 수확 기술의 개발이 필요하다.

깻잎은 단일 줄기형 구조를 가지며, 마디마다 교차 형태로 잎이 자라난다. 작업자가 줄기를 잡고 잎자루를 감지하여 수확하는 것을 로봇팔로 대체하기 위해서는 한 손 동작을 모사할 수 있는 단일 로봇팔 기반 엔드이펙터 설계가 필요하다. 기존의 일반 그리퍼는 잎 표면을 직접 파지해 손상을 유발하므로, 줄기와 잎자루만 선택적으로 감지‧절단하는 엔드이펙터가 요구된다.

<img width="571" height="215" alt="image" src="https://github.com/user-attachments/assets/0aee7291-d2c3-40a9-a609-035812ecdbf4" />

본 연구의 목적은 깻잎의 연약한 잎이 손상되지 않으면서도 효율적이고 정밀한 수확이 가능한 자동화 시스템을 구축하는 데 있다. 포토센서와 마이크로 포토센서를 활용한 비접촉식 줄기·잎자루 감지 기술과 와이어 구동 절단 메커니즘을 결합한 집게형 엔드이펙터를 설계‧제작하였으며, 잎의 방향성과 기울기에 따른 절단 위치를 정밀하게 인식하기 위해 객체의 회전 정보를 포함한 Oriented Bounding Box(OBB) 기반 영상 인식 기술을 적용하여 잎의 손상을 최소화하고, 로봇팔의 절단 경로를 최적화하여 안정적이고 반복 가능한 수확 작업이 가능한 기반 기술을 확립하고자 한다.
