## Project Background
깻잎은 연중 수확이 가능한 주요 시설 엽채류이며, 현재까지도 대부분의 수확 과정이 숙련 인력의 수작업에 의존하고 있다. 장시간 반복으로 인한 신체적 피로와 인력 부족문제가 있으며, 효율적이고 정밀한 자동 수확 기술의 개발이 필요하다.

깻잎은 단일 줄기형 구조를 가지며, 마디마다 교차 형태로 잎이 자라난다. 작업자가 줄기를 잡고 잎자루를 감지하여 수확하는 것을 로봇팔로 대체하기 위해서는 한 손 동작을 모사할 수 있는 단일 로봇팔 기반 엔드이펙터 설계가 필요하다. 기존의 일반 그리퍼는 잎 표면을 직접 파지해 손상을 유발하므로, 줄기와 잎자루만 선택적으로 감지‧절단하는 엔드이펙터가 요구된다.

<img width="571" height="215" alt="image" src="https://github.com/user-attachments/assets/0aee7291-d2c3-40a9-a609-035812ecdbf4" />

본 연구의 목적은 깻잎의 연약한 잎이 손상되지 않으면서도 효율적이고 정밀한 수확이 가능한 자동화 시스템을 구축하는 데 있다. 포토센서와 마이크로 포토센서를 활용한 비접촉식 줄기·잎자루 감지 기술과 와이어 구동 절단 메커니즘을 결합한 집게형 엔드이펙터를 설계‧제작하였으며, 잎의 방향성과 기울기에 따른 절단 위치를 정밀하게 인식하기 위해 객체의 회전 정보를 포함한 Oriented Bounding Box(OBB) 기반 영상 인식 기술을 적용하여 잎의 손상을 최소화하고, 로봇팔의 절단 경로를 최적화하여 안정적이고 반복 가능한 수확 작업이 가능한 기반 기술을 확립하고자 한다.

## System Model

### 전체 구조 개요
본 시스템은 비전 인식부, 로봇 제어부, 엔드이펙터 구동부의 세 가지 핵심 모듈로 구성된다.
1. 비전 인식부
    - RGB-D 카메라(RealSense L515)를 이용해 깻잎의 컬러 및 깊이 정보를 수집
    - YOLOv11 OBB 모델을 통해 깻잎의 위치 및 회전각 탐지
    - 픽셀 좌표 → 3D 좌표 변환 후 로봇 좌표계 보정
2. 로봇 제어부
    - Jetson Orin Nano에서 인식 결과를 처리 후 TCP/IP로 로봇 제어 신호 전달
    - Neuromeka Indy7 기반 6자유도 로봇팔 제어
    - 엔드이펙터의 진입, 파지, 절단 궤적 자동 수행
3. 엔드이펙터 구동부
    - 포토센서: 줄기 감지
    - 마이크로 포토센서: 잎자루 감지
    - Dynamixel + 와이어 메커니즘: 절단 및 파지
    - OpenCR 보드로 제어 및 신호 통합
    - RS485-TTL 컨버터 적용으로 장거리 통신 안정성 확보

### 작동 절차
깻잎 자동 수확 시스템의 전체 동작 절차는 5단계로 구성되어 있다.

1. 목표 잎 탐지 (Target Detection)
    - RGB-D 카메라로 촬영한 영상에서 YOLOv11-OBB 모델이 깻잎의 위치와 회전각을 검출
    - 격자 형태로 재배된 깻잎의 좌표를 좌하단 → 우상단 순서로 정렬하여 수확 경로를 설정
2. 측면 진입 (Lateral Approach)
    - 인식된 3D 좌표를 기반으로 로봇팔이 잎의 측면으로 접근
    - 잎의 생장점을 보호하기 위해 평균 6 cm 아래에서 진입
3. 줄기 탐지 및 파지 (Stem Detection & Grasping)
    - 포토센서가 줄기의 통과를 감지하면 집게부가 닫히며 줄기를 안정적으로 파지
4. 잎자루 감지 및 절단 준비 (Petiole Detection)
    - 줄기를 따라 상승하면서 마이크로 포토센서가 잎자루 통과 신호를 인식
    - 절단 위치 정렬 후 와이어 장력을 준비
5. 절단 및 수확 완료 (Cutting & Harvesting)
    - 와이어 구동 메커니즘이 작동하여 잎자루 절단과 파지를 동시에 수행
    - 수확 완료 후 스프링 복원력으로 원위치 복귀 → 다음 잎으로 이동
